# STEALTH DETECTION

## 프로젝트 소개

## Pygame을 활용하여 제작한 2D 잠입 액션 게임입니다. 플레이어는 감시 카메라(CCTV)와 경비원의 시야를 피해 보물을 획득해야 합니다.

## 개발 환경

* Python 3.x
* Pygame

## 실행 방법

1. 라이브러리 설치
    pip install -r requirements.txt
2. 게임 실행
    python main.py

## 조작 방법

* 이동 : W, A, S, D 또는 방향키
* 총 사용 : 마우스 왼쪽 클릭
* 난이도 선택 : 1(EASY), 2(NORMAL), 3(HARD)
* 게임 시작 : SPACE
* 결과 화면 : R 키

## 주요 기능

* CCTV 시야 감지 시스템
* 경비원 순찰 및 시야 판정
* 레이캐스팅 기반 시야 차단
* 프리즈 아이템
* 총기 아이템
* 미니맵
* 최고 기록 저장 기능
